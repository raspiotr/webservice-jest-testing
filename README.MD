### Webservice with testing (Node.js, Express, Jest)

**Simulating a simple webservice and unit-tests using "Jest"**

---

### Supported Routes:

- **GET /users** - Returns an array of all users.
- **GET /users/:id** - If such an 'id' exists, returns a user object.
